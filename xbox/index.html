XCM White 360 HDMI 1080p Cable

This item seems to be creating quite the stir for something that is just a retractable HDMI cable, which is cool as it is. But this is not what it is being reported as but a cable allowing a standard Xbox 360 to be connected using HDMI, something only the Elite version of the console has (it has a HDMI connector). This cable does not allow some magic conversion from Xbox 360 Core / Standard output to HDMI.

Well done for a first a retractable HDMI cable, but this is not a 360 - HDMI cable just a HDMI to HDMI cable. It happens to be white, this is where the misunderstanding has happened as the Core / Standard console is white so a cable described as "White 360" could be mis-read, however the picture also featured shows quite clearly the same connection on both ends which any converter cable would not have, plus the round part is where the cable retracts into!
